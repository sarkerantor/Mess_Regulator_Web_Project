🍽️ Mess Regulator
📌 Project Overview

Mess Management System is a web-based application designed to manage student mess activities in a smart and organized way.
It helps track daily meals, manage expenses, calculate monthly costs, and maintain clear financial records.

This project was developed as part of our Web Engineering course.

🚀 Live Demo

🔗 https://antor.kesug.com

🎯 Project Goals

Automate meal tracking and cost calculation

Minimize manual mistakes

Save time in daily management

Maintain financial transparency

Provide a simple and digital solution for mess management

✨ Key Features

✅ Member Registration System

✅ Daily Meal Entry

✅ Expense Management
